# 👨‍💻 Portfolio Personnel - Julian SEILLER

Bienvenue sur le dépôt de mon portfolio web interactif ! Ce site a été conçu pour présenter mon parcours, mes compétences techniques (BTS SIO option SLAM) et mes réalisations professionnelles.

🌍 **Voir le site en ligne :** [portfolio-juliansllr.vercel.app](https://portfolio-juliansllr.vercel.app)

---

## 🚀 Fonctionnalités

* **Bilingue (FR / EN) :** Le site propose un basculement fluide entre la version française et anglaise.
* **Génération Dynamique :** Les projets, les réalisations professionnelles et les cartes de compétences sont générés dynamiquement via JavaScript (fichiers `projets.js` et `skillCards.js`).
* **Design Responsive :** L'interface s'adapte à tous les écrans, avec un menu "hamburger" personnalisé pour les supports mobiles.
* **Animations Fluides :** Utilisation de la librairie ScrollReveal pour animer l'apparition des éléments au défilement.
* **Formulaire de Contact :** Intégration d'un formulaire redirigeant automatiquement vers une interface mail.

---

## 🛠️ Stack Technique

Ce projet est volontairement développé sans framework lourd afin de consolider mes acquis sur les bases du web :

* **Langages :** HTML5, CSS3, JavaScript (Vanilla)
* **Librairies tierces :** ScrollReveal (Animations)
* **Polices & Icônes :** Google Fonts (Gabarito), FontAwesome, Devicon
* **Hébergement :** Vercel

---

## 📁 Structure du Projet

Voici l'organisation des fichiers de ce dépôt :

* `/assets/` : Dossier contenant toutes les ressources statiques.
  * `/files/` : CV en format PDF et fiches de réalisations (FRP).
  * `/icons/` & `/img/` : Images, logos et icônes SVG/PNG.
  * `/javascript/` : Scripts gérant la logique du site (`main.js`, `projets.js`, `skillCards.js`).
* `index.html` : Page principale en version française.
* `indexEN.html` : Page principale en version anglaise.
* `styles.css` : Feuille de style globale du portfolio.

---

## 💻 Installation en local

Si vous souhaitez explorer le code source ou lancer le projet sur votre machine :

1. Clonez ce dépôt :
   git clone https://github.com/JulianSLLR/Portfolio.git

2. Ouvrez le dossier du projet :
   cd Portfolio

3. Ouvrez simplement le fichier `index.html` dans votre navigateur web préféré. Aucun serveur local ni installation de dépendances n'est requis !
